## 前付け

Python のインストールは[Anaconda](https://www.anaconda.com/distribution/)をオススメする。
インストール出来たら、`environment.yml`を使って必要なパッケージをインストールする。

    conda env create -f environment.yml

その後、新しい環境をアクティベートする。

    conda activate onkyo100pon
