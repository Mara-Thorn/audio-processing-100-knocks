## 前付け

パイソンのインストールは[Anaconda](https://www.anaconda.com/distribution/)がオススメ。
インストール出来たら、`environment.yml`を使って必要なパッケージをインストールする。

    conda env create -f environment.yml

インストール後に、新しいエンバイロンメント変換

    conda activate onkyo100pon
